Konsep Dasar Fungsi

Definisi Fungsi

Fungsi adalah suatu aturan atau relasi yang menghubungkan setiap elemen dalam domain (daerah asal) dengan tepat satu elemen dalam kodomain (daerah kawan)

f: A → B atau y = f(x)

Komponen Fungsi

Domain (A) = Himpunan semua nilai input yang valid (daerah asal)
Kodomain (B) = Himpunan yang memuat semua kemungkinan output (daerah kawan)
Range = Himpunan nilai output yang benar-benar dihasilkan (daerah hasil)

Contoh Fungsi

f(x) = 2x + 1 dengan domain {1, 2, 3}

Komposisi Fungsi

Kalkulator Komposisi

Masukkan fungsi dan nilai x untuk melihat langkah-langkah komposisi

Jenis Komposisi

(f ∘ g)(x) = f(g(x))

Komposisi f dengan g: terapkan g terlebih dahulu, lalu f

Contoh:

f(x) = 2x + 1, g(x) = x²

(f ∘ g)(x) = f(x²) = 2(x²) + 1 = 2x² + 1

(g ∘ f)(x) = g(f(x))

Komposisi g dengan f: terapkan f terlebih dahulu, lalu g

Contoh:

f(x) = 2x + 1, g(x) = x²

(g ∘ f)(x) = g(2x + 1) = (2x + 1)² = 4x² + 4x + 1

Fungsi Linear

f(x) = 3x - 5

1. y = 3x - 5

2. x = 3y - 5

3. 3y = x + 5

4. y = (x + 5)/3

f⁻¹(x) = (x + 5)/3

Fungsi Kuadrat

f(x) = x² (x ≥ 0)

1. y = x²

2. x = y²

3. y = ±√x

4. Karena x ≥ 0, maka y ≥ 0

f⁻¹(x) = √x

Aplikasi Fungsi dalam Kehidupan

⚡ Fisika - Gerak dan Kinematika

Rumus: s(t) = s₀ + v₀t + ½at²

Fungsi untuk menggambarkan posisi, kecepatan, dan percepatan

h(1) = 2 + 20(1) + ½(-10)(1)² = 17 meter

💰 Ekonomi - Fungsi Biaya dan Keuntungan

Rumus: P(x) = R(x) - C(x)

Model matematika untuk analisis bisnis dan ekonomi

P(50) = 100(50) - (1000 + 50(50)) = 1500 ribu

🦠 Biologi - Pertumbuhan Populasi

Rumus: P(t) = P₀eʳᵗ

Model pertumbuhan eksponensial organisme

P(5) = 100 × e^(0.1×5) ≈ 165 organisme
