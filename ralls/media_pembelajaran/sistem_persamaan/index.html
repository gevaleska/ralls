SPLDV (Sistem Persamaan Linear Dua Variabel)

Definisi:

Sistem persamaan yang terdiri dari dua persamaan linear dengan dua variabel (x, y)

Bentuk Umum:

ax + by = c
dx + ey = f

Metode Penyelesaian:

  • Grafik: Menggambar garis dan mencari titik potong
  • Substitusi: Mengganti nilai satu variabel
  • Eliminasi: Menghilangkan satu variabel
  • Campuran: Kombinasi eliminasi dan substitusi

SPLTV (Sistem Persamaan Linear Tiga Variabel)

Definisi:

Sistem persamaan yang terdiri dari tiga persamaan linear dengan tiga variabel (x, y, z)

Bentuk Umum:

ax + by + cz = d
ex + fy + gz = h
ix + jy + kz = l

Metode Penyelesaian:

  • Substitusi: Mengganti nilai variabel secara bertahap
  • Eliminasi: Menghilangkan variabel satu per satu
  • Campuran: Kombinasi eliminasi dan substitusi
  • Matriks: Menggunakan operasi matriks

📝 Contoh Soal

Contoh Soal SPLDV

Masalah Belanja

Soal: Ani membeli 3 kg apel dan 2 kg jeruk seharga Rp 79.000. Budi membeli 2 kg apel dan 3 kg jeruk seharga Rp 71.000. Berapa harga per kg apel dan jeruk?

Langkah-langkah:

Misalkan: x = harga apel per kg, y = harga jeruk per kg

Persamaan 1: 3x + 2y = 79000

Persamaan 2: 2x + 3y = 71000

Metode Eliminasi:

Eliminasi x: kalikan persamaan 1 dengan 2, persamaan 2 dengan 3

6x + 4y = 158000

6x + 9y = 213000

Kurangkan: -5y = -55000 → y = 11000

Substitusi y = 11000 ke persamaan 1:

3x + 2(11000) = 79000

3x = 57000 → x = 19000

Jawaban: Harga apel Rp 19.000/kg, harga jeruk Rp 11.000/kg

Masalah Umur

Soal: Umur ayah 3 kali umur anak. Lima tahun lagi, umur ayah akan menjadi 2 kali umur anak. Tentukan umur mereka sekarang.

Langkah-langkah:

Misalkan: x = umur ayah sekarang, y = umur anak sekarang

Persamaan 1: x = 3y

Persamaan 2: x + 5 = 2(y + 5)

Metode Substitusi:

Dari persamaan 1: x = 3y

Substitusi ke persamaan 2:

3y + 5 = 2(y + 5)

3y + 5 = 2y + 10

y = 5

Maka x = 3(5) = 15

Jawaban: Umur ayah 15 tahun, umur anak 5 tahun

Contoh Soal SPLTV

Masalah Investasi

Soal: Seorang investor menanamkan uangnya dalam 3 jenis investasi: saham (x), obligasi (y), dan deposito (z). Total investasi Rp 100 juta. Keuntungan saham 8%, obligasi 6%, deposito 4% dengan total keuntungan Rp 6.2 juta. Investasi saham 2 kali investasi obligasi. Berapa masing-masing investasi?

Langkah-langkah:

Persamaan 1: x + y + z = 100000000

Persamaan 2: 0.08x + 0.06y + 0.04z = 6200000

Persamaan 3: x = 2y

Penyelesaian:

Substitusi x = 2y ke persamaan 1 dan 2:

2y + y + z = 100000000 → 3y + z = 100000000

0.16y + 0.06y + 0.04z = 6200000 → 0.22y + 0.04z = 6200000

Dari persamaan pertama: z = 100000000 - 3y

Substitusi: 0.22y + 0.04(100000000 - 3y) = 6200000

0.22y + 4000000 - 0.12y = 6200000

0.1y = 2200000 → y = 22000000

Maka: x = 2(22000000) = 44000000

z = 100000000 - 44000000 - 22000000 = 34000000

Jawaban: Saham Rp 44 juta, Obligasi Rp 22 juta, Deposito Rp 34 juta
