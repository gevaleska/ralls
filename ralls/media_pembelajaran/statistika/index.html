Pengenalan Statistika

Definisi Statistika

Statistika adalah ilmu yang mempelajari cara mengumpulkan, mengolah, menganalisis, dan menyajikan data untuk penarikan kesimpulan yang berguna dalam pengambilan keputusan.

Jenis-jenis Data

Data Kualitatif

Data yang berbentuk kategori atau atribut. Contoh: jenis kelamin, warna, kualitas produk.

Data Kuantitatif

Data yang berbentuk angka. Dibagi menjadi:

  • Diskrit: Data yang nilainya berupa bilangan bulat. Contoh: jumlah siswa, jumlah mobil.
  • Kontinu: Data yang nilainya berupa interval. Contoh: tinggi badan, berat badan.

Populasi dan Sampel

Populasi: Keseluruhan objek yang akan diteliti.

Sampel: Bagian dari populasi yang dipilih untuk diteliti.

  • Diagram Batang: Untuk data kualitatif atau kuantitatif diskrit
  • Diagram Lingkaran: Untuk menunjukkan proporsi data
  • Histogram: Untuk data kuantitatif berkelompok
  • Poligon Frekuensi: Garis yang menghubungkan titik-titik tengah histogram

Ukuran Pemusatan Data

Mean (Rata-rata)

Jumlah semua nilai dibagi dengan banyaknya data.

$$\bar{x} = \frac{\sum x_i}{n}$$

Data berkelompok:

$$\bar{x} = \frac{\sum f_i \cdot x_i}{\sum f_i}$$

dimana fi = frekuensi, xi = titik tengah kelas

Median (Nilai Tengah)

Nilai yang membagi data menjadi dua bagian sama besar setelah data diurutkan.

Data ganjil (n ganjil):

$$Me = x_{\frac{n+1}{2}}$$

Data genap (n genap):

$$Me = \frac{x_{\frac{n}{2}} + x_{\frac{n}{2}+1}}{2}$$

Modus

Nilai yang paling sering muncul dalam data.

Untuk data berkelompok, modus berada pada kelas dengan frekuensi tertinggi.

Ukuran Penyebaran Data

Jangkauan (Range)

Selisih antara nilai tertinggi dan nilai terendah.

$$R = x_{max} - x_{min}$$

Ragam (Variance)

Rata-rata kuadrat selisih setiap data dengan rata-ratanya.

$$s^2 = \frac{\sum (x_i - \bar{x})^2}{n-1}$$

Simpangan Baku (Standard Deviation)

Akar kuadrat dari ragam.

$$s = \sqrt{s^2} = \sqrt{\frac{\sum (x_i - \bar{x})^2}{n-1}}$$

Kuartil dan Persentil

Kuartil: Nilai yang membagi data menjadi 4 bagian sama besar.

  • Q1 = kuartil pertama (25%)
  • Q2 = kuartil kedua = median (50%)
  • Q3 = kuartil ketiga (75%)

Contoh Soal dan Pembahasan

Contoh 1: Menghitung Mean Data Tunggal

Soal: Nilai ujian 8 siswa adalah: 75, 80, 85, 70, 90, 85, 75, 88. Hitunglah rata-rata nilai ujian tersebut!

Pembahasan:

Langkah 1: Jumlahkan semua nilai

75 + 80 + 85 + 70 + 90 + 85 + 75 + 88 = 648

Langkah 2: Hitung banyaknya data

n = 8

Langkah 3: Gunakan rumus mean

$$\bar{x} = \frac{\sum x_i}{n} = \frac{648}{8} = 81$$

Jawab: Rata-rata nilai ujian adalah 81

Contoh 2: Menghitung Median

Soal: Dari data: 12, 15, 10, 18, 14, 16, 11. Tentukan median!

Pembahasan:

Langkah 1: Urutkan data dari kecil ke besar

10, 11, 12, 14, 15, 16, 18

Langkah 2: Tentukan banyaknya data

n = 7 (ganjil)

Langkah 3: Cari posisi median

Posisi median = (n+1)/2 = (7+1)/2 = 4

Langkah 4: Tentukan nilai median

Median = data ke-4 = 14

Jawab: Median = 14

Contoh 3: Membuat Tabel Distribusi Frekuensi

Soal: Data nilai ujian 20 siswa: 65, 70, 75, 80, 68, 72, 78, 85, 67, 73, 76, 82, 69, 74, 77, 83, 66, 71, 79, 84. Buatlah tabel distribusi frekuensi dengan 5 kelas!

Pembahasan:

Langkah 1: Tentukan nilai tertinggi dan terendah

Nilai tertinggi = 85, Nilai terendah = 65

Langkah 2: Hitung jangkauan (range)

R = 85 - 65 = 20

Langkah 3: Tentukan panjang interval

Panjang interval = R/banyak kelas = 20/5 = 4

🏢 Bisnis dan Ekonomi

Analisis Penjualan

Toko ABC mencatat penjualan harian selama sebulan. Dengan statistika, pemilik dapat mengetahui rata-rata penjualan, hari dengan penjualan tertinggi, dan tren penjualan.

Kontrol Kualitas Produksi

Pabrik menggunakan statistika untuk memantau kualitas produk, mengidentifikasi cacat produksi, dan memastikan standar kualitas terpenuhi.

Riset Pasar

Survei konsumen menggunakan statistika untuk menganalisis preferensi pelanggan dan membantu pengambilan keputusan bisnis.

🏠 Kehidupan Sehari-hari

Prakiraan Cuaca

BMKG menggunakan data statistik suhu, kelembaban, dan tekanan udara untuk memprediksi cuaca esok hari.

Statistik Olahraga

Rata-rata skor pemain, persentase tembakan berhasil, dan analisis performa tim menggunakan konsep statistika.

Data Kesehatan

Rumah sakit menganalisis data pasien untuk mengidentifikasi penyakit yang sering muncul dan merencanakan pelayanan kesehatan.

🏭 Aplikasi Industri
