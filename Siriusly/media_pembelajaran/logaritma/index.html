Konsep Dasar Logaritma

Definisi Logaritma

Logaritma adalah operasi matematika yang merupakan kebalikan dari eksponen atau pemangkatan

log_a(b) = c ↔ a^c = b

Komponen Logaritma

a = Basis logaritma (bilangan pokok)
b = Numerus (argumen/bilangan yang dicari logaritmanya)
c = Hasil logaritma (pangkat)

Jenis-jenis Logaritma

log(x) atau log₁₀(x)

Logaritma Biasa/Briggs

Logaritma dengan basis 10, sering digunakan dalam sains dan teknik

Contoh: log(100) = 2 karena 10² = 100

ln(x) atau log_e(x)

Logaritma Natural/Nepper

Logaritma dengan basis e ≈ 2.718, banyak digunakan dalam kalkulus

Contoh: ln(e) = 1 karena e¹ = e

log₂(x)

Logaritma Biner

Logaritma dengan basis 2, digunakan dalam ilmu komputer

Contoh: log₂(16) = 4 karena 2⁴ = 16

log_a(x)

Logaritma Umum

Logaritma dengan basis sembarang a > 0, a ≠ 1

Contoh: log₃(27) = 3 karena 3³ = 27

Sifat-Sifat Logaritma

Pilih Sifat untuk Dipelajari

Logaritma Basis Terhadap Dirinya Sendiri

log_a(a) = 1

Logaritma suatu bilangan terhadap dirinya sendiri selalu bernilai 1

Contoh:

log₅(5) = 1, log₁₀(10) = 1

Bukti:

Karena a¹ = a, maka log_a(a) = 1

Operasi dengan Logaritma

➕ Penjumlahan Logaritma

log_a(x) + log_a(y) = log_a(xy)

Penjumlahan logaritma dengan basis sama

log ( ) + log ( )
log₂(4) + log₂(8) = log₂(32) = 5

➖ Pengurangan Logaritma

log_a(x) - log_a(y) = log_a(x/y)

Pengurangan logaritma dengan basis sama

log ( ) - log ( )
log₃(81) - log₃(9) = log₃(9) = 2

✖️ Perubahan Basis

log_a(x) = log_b(x) / log_b(a)

Mengubah basis logaritma

log ( ) = log (...)
log₃(10) = log₁₀(10) / log₁₀(3) ≈ 2.096

⚡ Pangkat Logaritma

log_a(x^n) = n × log_a(x)

Logaritma dari bilangan berpangkat

log ( ^ )
log₂(5³) = 3 × log₂(5) ≈ 6.966

log_a(x) + log_a(y) = c

Metode: Gunakan sifat perkalian: log_a(xy) = c → xy = a^c

Contoh dan Latihan

Contoh: log₂(x) + log₂(3) = 5

2. 3x = 2⁵ = 32

Aplikasi Logaritma dalam Kehidupan

🧪 Skala pH dalam Kimia

Rumus: pH = -log₁₀[H⁺]

Mengukur tingkat keasaman atau kebasaan larutan

Skala: 0-14 (asam: 0-7, netral: 7, basa: 7-14)

pH = -log₁₀(0.01) = -(-2) = 2 (sangat asam)

🌍 Skala Richter untuk Gempa

Rumus: M = log₁₀(A/A₀)

Mengukur kekuatan/magnitudo gempa bumi

Catatan: Setiap peningkatan 1.0 = amplitudo 10x lebih besar

M = log₁₀(1000/1) = log₁₀(1000) = 3.0

🔊 Skala Desibel untuk Suara

Rumus: dB = 10 × log₁₀(I/I₀)

Mengukur intensitas atau kenyaringan suara

Referensi: Normal (60 dB), Konser (110 dB), Jet (140 dB)

dB = 10 × log₁₀(0.001/10⁻¹²) = 90 dB

💰 Bunga Majemuk dalam Keuangan

Rumus: t = log(A/P) / log(1+r)

Menghitung waktu investasi untuk mencapai target tertentu

Aplikasi: Perencanaan keuangan, investasi, kredit

t = log(20M/10M) / log(1.08) ≈ 9 tahun

🦠 Pertumbuhan Populasi

Rumus: t = log(N/N₀) / log(1+r)

Model pertumbuhan eksponensial organisme

Aplikasi: Ekologi, demografi, epidemiologi

t = log(10000/1000) / log(1.1) ≈ 24.2 periode
